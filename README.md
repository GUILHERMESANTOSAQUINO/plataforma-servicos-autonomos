# Plataforma Digital para Divulgação de Serviços Autônomos

Este projeto é uma aplicação web inicial desenvolvida para divulgar serviços prestados por trabalhadores autônomos do bairro Santa Terezinha, em Aquidauana/MS.

A proposta busca organizar informações básicas sobre os profissionais, como nome, tipo de serviço, descrição, região de atendimento, horário disponível e contato.

## Objetivo

Desenvolver uma aplicação web simples e responsiva para facilitar a consulta e a divulgação de serviços autônomos da comunidade.

## Funcionalidades

- Exibição de serviços por categorias;
- Filtro de serviços por tipo de atividade;
- Cards com informações dos trabalhadores autônomos;
- Formulário para simulação de cadastro de novos serviços;
- Layout responsivo para celulares e computadores.

## Tecnologias utilizadas

- HTML5;
- CSS3;
- JavaScript;
- Bootstrap 5.

## Como executar o projeto

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto.
3. Clique duas vezes no arquivo `index.html`.
4. A aplicação será aberta no navegador.

## Observação

Esta é uma versão inicial da aplicação. Os dados cadastrados pelo formulário são adicionados apenas durante o uso da página e não ficam salvos após atualizar ou fechar o navegador.
